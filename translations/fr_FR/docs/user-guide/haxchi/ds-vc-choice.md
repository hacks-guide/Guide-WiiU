# Haxchi {docsify-ignore-all}

## Choisissez votre jeu DS VC

La première chose à faire est de décider quel jeu Console Virtuelle DS ("VC" pour Virtual Console en anglais) vous allez acheter.  
Notez que le jeu dans lequel vous aurez injecté Haxchi/CBHC ne sera plus jouable tant que vous n'aurez pas désinstallé Haxchi/CBHC et réinstallé le jeu depuis l'eShop.
