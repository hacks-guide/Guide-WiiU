# vWii Modding Guide {docsify-ignore-all}

## Installing cIOSes

1. Power on your Wii U and launch vWii.
1. Launch The Homebrew Channel.
1. Launch d2x cIOS Installer.
1. Set everything to match the following:
    - Select cIOS: `d2x-v11-beta1-vWii`
    - Select cIOS base: `56`
    - Select cIOS slot: `249`
1. Press the `A` button to install.
1. Set everything to match the following:
    - Select cIOS: `d2x-v11-beta1-vWii`
    - Select cIOS base: `57`
    - Select cIOS slot: `250`
1. Press the `A` button to install.
1. Set everything to match the following:
    - Select cIOS: `d2x-v11-beta1-vWii`
    - Select cIOS base: `58`
    - Select cIOS slot: `251`
1. Press the `A` button to install.
1. Press the `B` button to exit.

!> If you are only getting `d2x-v6` or `d2x-v11-beta1`, redownload the app and/or make sure your SD card is unlocked.
