# Hachage des fichiers

Cette page fournit une liste de valeurs de hachage de fichiers connus pour les liens de téléchargement présents dans ce guide, ainsi que des instructions sur comment vérifier ces hachages par rapport à vos fichiers téléchargés. Cela pourrait vous aider si vous semblez rencontrer des problèmes inconnus où un logiciel téléchargé ne se comporte pas comme vous l'attendriez.

## Vérifier les hachages

### Section I - Prérequis

- Un navigateur web moderne, de préférence sur un OS de bureau
- [Online SHA256 Checksum Tool](https://emn178.github.io/online-tools/sha256_checksum.html)

### Section II - Instructions

::: tip

Cette section utilise **wudd** comme exemple, et devrait correspondre au hachage SHA256 fourni ci-dessous.

:::

1. Ouvrez l'utilitaire de somme de contrôle (checksum tool) fourni ci-dessus dans le navigateur web de votre choix.

2. Sur la page, glissez et déposer le fichier que vous souhaitez vérifier dans la boîte Input.

   ![](/assets/img/hash-checker/drag-file.png)

3. Copiez la valeur de hachage depuis la boite Output, qui a été calculée à partir de votre fichier.

   ![](/assets/img/hash-checker/copy-hash.png)

4. Sur cette page, faites un `CTRL + F` ou la combinaison de touches appropriée pour ouvrier la fonction de recherche dans la page de votre navigateur. Vous pouvez copier le hachage là, qui devrait correspondre **EXACTEMENT** au hachage fourni pour le fichier que vous souhaitiez télécharger.

   ![](/assets/img/hash-checker/check-hash.png)
