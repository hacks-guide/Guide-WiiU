# Désinstaller CBHC

Cette page vous guidera tout au long du processus de désinstallation de CBHC de votre Wii U.

## Ce dont vous avez besoin

- La dernière version du [Homebrew Launcher](https://github.com/dimok789/homebrew_launcher/releases/tag/v1.5)
- Le [CBHC installer](https://github.com/FIX94/haxchi/releases)

## Instructions

1. Lancez le Homebrew Launcher.
2. Lancez CBHC Installer.
3. Sélectionnez le jeu Console Virtuelle DS dont vous voulez désinstaller CBHC puis appuyez sur le bouton `A`.
4. Lisez les avertissements puis appuyez sur le bouton `B`.
5. Vérifiez que votre Wii U ne démarre plus automatiquement sur CBHC en redémarrant la console.
6. Si l'écran de démarrage automatique (Autobooting...) et/ou de menu ne sont pas apparus lors du démarrage de la console, vous pouvez lancer les Paramètres de la console, puis aller dans la Gestion des données et ensuite désinstaller en toute sécurité l'application appelée `DON'T TOUCH ME`.

::: tip

Si vous recevez une pop-up indiquant "Une erreur de mémoire système est survenue. Éteignez la console, puis réessayez." pendant la suppression de l'application `DON'T TOUCH ME`, redémarrez la console et essayez à nouveau de supprimer l'application. **Veuillez** vous assurer que vous avez suivi toutes les étapes ci-dessus avant de supprimer l'application.

:::
