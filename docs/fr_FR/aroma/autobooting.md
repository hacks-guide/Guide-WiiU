# Autodémarrer Aroma

Actuellement, à chaque fois que vous voudrez lancer Aroma, vous devrez lancer l'appli Informations sur la santé et la sécurité. Si vous souhaitez lancer automatiquement Aroma au démarrage de la console, vous pouvez démarrer automatiquement l'appli Informations sur la santé et la sécurité.

Si vous ne voulez pas autodémarrer Aroma, vous pouvez passer cette étape et suivre la section ci-dessous.

## Instructions

1. Démarrez la console pour arriver sur le menu Wii U, puis lancez l'appli Informations sur la santé et la sécurité.
2. Appuyez sur A pour lancer l'environnement `aroma`.
3. Appuyez sur A pour lancer le menu Wii U.
4. Une fois que vous êtes sur le menu Wii U, lancez le PayloadLoader Installer.
5. Appuyez sur A pour sélectionner `Check`.
6. Sélectionnez `Boot options`.
7. Il vous sera demandé si vous souhaitez changer le titre de démarrage. Appuyez sur A pour sélectionner `Switch to PayloadLoader` (passer à PayloadLoader).
8. Quand le processus est terminé, appuyez sur A pour éteindre la console.
9. Le PayloadLoader sera maintenant lancé automatiquement à chaque démarrage de la console.

## Mettre en place PayloadLoader, Environment Loader et Aroma

Maintenant, nous allons faire démarrer automatiquement l'environnement Aroma lorsque la console charge l'appli Informations sur la santé et la sécurité et sélectionner le menu Wii U comme l'option de démarrage par défaut.

1. Lancez l'EnvironmentLoader.
   - Si vous autodémarrez le PayloadLoader, allumez simplement votre Wii U.
   - Si vous avez passé les étapes d'autodémarrage, lancez l'appli Informations sur la santé et la sécurité.
2. Appuyez sur Y sur `aroma` pour le définir comme votre environnement par défaut, puis appuyez sur A pour lancer Aroma.
   ![](/assets/img/guide/EL_Highlight.png)
   - Pour ouvrir l'Environnement Loader à l'avenir, vous devrez maintenir X lorsque votre Wii U démarre ou charge l'appli Informations sur la santé et la sécurité.
3. Dans le Boot Selector d'Aroma, le `Wii U Menu` devrait être déjà sélectionné. Appuyez sur Y pour le définir comme l'option d'autodémarrage par défaut, puis appuyez sur A pour lancer le menu Wii U.
   ![](/assets/img/guide/ABM_Highlight.png)
4. Aroma sera désormais lancé à chaque fois que vous démarrerez la console (ou chargez Santé et Sécurité) puis directement dans le menu Wii U.
   - Pour ouvrir le sélecteur de démarrage d'Aroma dans le futur, vous devrez maintenir START (+) pendant que la console s'allume ou charge l'appli Informations sur la santé et la sécurité.
   - Naviguez dans la liste en utilisant la croix directionnelle pour survoler le titre que vous voulez démarrer automatiquement, puis appuyez sur le bouton Y pour définir le titre qui sera lancé automatiquement au démarrage de la console.
   - Appuyez sur A pour lancer le titre sélectionné.
